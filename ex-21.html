How much does semantic HTML improves the foundation of any web page?

  • HTML5 is the version of HTML currently used today
  • A div element and a span element are used to structure content,but neither of these give meaning to either the programmer or the browser
  • One of the goals of HTML5 is to upgrade the previous structure with new semantic elements that provide better benefits:
    • It can be accessed and engaged with by various users,such as ones who use screen readers
    • The content in these elements are more discoverable for user through places like search engines. Sucha as Google
    • The actual HTML code becomes cleaner and better readability for programmers and easier to maintain
